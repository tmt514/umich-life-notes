## 5. C++ 語法複習

### 變數宣告與計算

#### 宣告一個整數
```
int a;
```

#### 宣告一個(雙倍精確)浮點數
```
double a;
```

#### 計算兩個數的算數平均
````
average = (a + b) / 2.0;
````
